1. Grid system

The grid system is a design tool that help and achieve visuel unity across Umbraco. The grid system is built on top of Twitter Bootstrap.

2. Section Bar

The section bar on the left side contain a avatar and 5 default sections: Content, Media, Users, Settings, Developer. Contains the bar more than 5 icons an extra icon appears. By clicking this a new section bar appear. Se animation. The avatar picture disappears and instead a back arrow appears.

Icon Description
Avatar. The login in users avatar.
Content. Click/Hover and the Content tree slides out.
Media. By clicking you access the Media section.
Users. By clicking you access the Users section.
Settings. By clicking you access the Settings section.
Developer. By clicking you access the Developer section.

3. Header

The header contain a minimal of information, to keep it simpel and clear to the user. Page name, witchs can easily been edit, just by clicking the text. Here you also have the ability to change language. In the right side of the header 2 buttons are showen, Preview and Publish. The preview button opens a new page with the current page. The Publish button is saving and publishing the current page. Umbraco is constantly saving the current document in the background, so you don't need to press save, but you can still access the Save button by clicking the arrow down, besides Publish. All header will always be visible to the user.

4. Tree

The tree is an hierarchical list of items related (and usually restricted) to a specific concept, which could be something like a content tree or a media tree.

The header of the tree contain a search function where you can easily access all content in Umbraco.

All icons in the tree are generated with a @font-face.

The tree contains tree states. Normal, hover and active. In hover and active state it is posible to access the context menu. The context menu contain the ablity to create a new document and do something else.
