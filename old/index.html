Javascript

Belle uses AngularJS as its javascript application framework, require.js for dependency handling and jquery + underscore for various bits and bobs.

Css 3

We use Twitter bootstrap for a clean CSS baseline and for having solid conventions for a responsive grid.On top, we add our own dash of color.

html 5

Belle uses HTML 5 and all the benefits you gain from it, and is expected to work in IE10+, Chrome, Safari and firefox.
