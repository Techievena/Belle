imageHelper (service in module umbraco.services )

Description

A helper object used for parsing image paths

Methods

  • detectIfImageByExtension(imagePath)

    Returns true/false, indicating if the given path has an allowed image extension

    Parameters

    • imagePath – {string} –

      Image path, ex: /media/1234/my-image.jpg

  • getImagePropertyValue(options)

    Returns the actual image path associated with the image property if there is one

    Parameters

    • options – {object} –

      Options object

    • options.imageModel – {object} –

      The media object to retrieve the image path from

  • getThumbnail(options)

    formats the display model used to display the content to the model used to save the content

    Parameters

    • options – {object} –

      Options object

    • options.imageModel – {object} –

      The media object to retrieve the image path from

  • getThumbnailFromPath(imagePath)

    Returns the path to the thumbnail version of a given media library image path

    Parameters

    • imagePath – {string} –

      Image path, ex: /media/1234/my-image.jpg

    Parameters

      Maximum width & height
