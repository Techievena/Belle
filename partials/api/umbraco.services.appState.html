appState (service in module umbraco.services )

Description

Tracks the various application state variables when working in the back office, raises events when state changes.

Methods

  • getGlobalState()

    Returns the current global state value by key - we do not return an object here - we do NOT want this to be publicly mutable and allow setting arbitrary values

  • getMenuState()

    Returns the current menu state value by key - we do not return an object here - we do NOT want this to be publicly mutable and allow setting arbitrary values

  • getSectionState()

    Returns the current section state value by key - we do not return an object here - we do NOT want this to be publicly mutable and allow setting arbitrary values

  • getTreeState()

    Returns the current tree state value by key - we do not return an object here - we do NOT want this to be publicly mutable and allow setting arbitrary values

  • setGlobalState()

    Sets a global state value by key

  • setMenuState()

    Sets a section state value by key

  • setSectionState()

    Sets a section state value by key

  • setTreeState()

    Sets a section state value by key
