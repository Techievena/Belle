logResource (service in module umbraco.resources )

Description

Retrives log history from umbraco

Methods

  • getEntityLog(id)

    Gets the log history for a give entity id

    usage

    logResource.getEntityLog(1234)
   .then(function(log) {
       alert('its here!');
   });

    Parameters

    • id – {Int} –

      id of entity to return log history

    Returns

    {Promise} –

    resourcePromise object containing the log.

  • getLog(type, since)

    Gets the log history for a given type of log entry

    usage

    logResource.getLog("save", new Date())
   .then(function(log) {
       alert('its here!');
   });

    Parameters

    • type – {String} –

      logtype to query for

    • since – {DateTime} –

      query the log back to this date, by defalt 7 days ago

    Returns

    {Promise} –

    resourcePromise object containing the log.

  • getUserLog(type, since)

    Gets the current users' log history for a given type of log entry
