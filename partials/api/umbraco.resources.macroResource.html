macroResource (service in module umbraco.resources )

Description

Methods

    Parameters

    • macroId – {int} –

      The macro id to get parameters for

    • pageId – {int} –

      The current page id

    • macroParamDictionary – {Array} –

      A dictionary of macro parameters
