treeService (service in module umbraco.services )

Description

The tree service factory, used internally by the umbTree and umbTreeItem directives

Methods

  • getMenuItemByAlias(args)

    Attempts to return a tree node's menu item based on the alias supplied, otherwise returns null.

    Parameters

    • args {object}

      An arguments object

    • args.treeNode {object}

      The tree node to get the menu item for

    • args.menuItemAlias {object}

      The menu item alias to attempt to find
