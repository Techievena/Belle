treeService (service in module umbraco.services )

Description

The tree service factory, used internally by the umbTree and umbTreeItem directives

Methods

  • getTreePackageFolder(treeAlias)

    Determines if the current tree is a plugin tree and if so returns the package folder it has declared so we know where to find it's views, otherwise it will just return undefined.

    Parameters

    • treeAlias – {String} –

      The tree alias to check

  • loadNodeChildren(args)

    Clears all node children, gets it's up-to-date children from the server and re-assigns them and then returns them in a promise.

    Parameters

    • args – {object} –

      An arguments object

    • args.node – {object} –

      The tree node

    • args.section – {object} –

      The current section
