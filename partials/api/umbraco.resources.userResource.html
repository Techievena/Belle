userResource (service in module umbraco.resources )

Description

Retrives user data from the server, cannot be used for authentication, for this, use the user.service

Methods

  • getAll()

    Gets all users available on the system

    usage

    contentResource.getAll()
   .then(function(userArray) {
       var myUsers = userArray; 
       alert('they are here!');
   });

    Returns

    {Promise} –

    resourcePromise object containing the user array.

  • getById(id)

    Gets a user with a given id

    usage

    userResource.getById(1234)
   .then(function(ent) {
       var myUser = ent; 
       alert('im here!');
   });

    Parameters

    • id {Int}

      id of user to return

    Returns

    {Promise} –

    resourcePromise object containing the user.
