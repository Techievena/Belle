umbModelMapper (service in module umbraco.services )

Description

Utility class to map/convert models

Usage

umbModelMapper();

Methods

  • convertToEntityBasic(source)

    Converts the source model to a basic entity model, it will throw an exception if there isn't enough data to create the model.

    Parameters

    • source – {Object} –

      The source model

    • source.id – {Number} –

      The node id of the model

    • source.name – {String} –

      The node name

    • source.icon – {String} –

      The models icon as a css class (.icon-doc)

    • source.parentId – {Number} –

      The parentID, if no parent, set to -1

    • source.path – {path} –

      comma-seperated string of ancestor IDs (-1,1234,1782,1234)
